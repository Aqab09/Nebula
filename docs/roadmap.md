# Nebula Roadmap

---

# Version 1.0 (MVP)

## Goal

Build a functional cloud operating platform capable of deploying Docker containers across multiple worker nodes.

## Features

- [ ] User Authentication
- [ ] Controller Service
- [ ] Worker Node Registration
- [ ] Heartbeat System
- [ ] Basic Scheduler
- [ ] Docker Deployment
- [ ] Dashboard
- [ ] PostgreSQL Integration
- [ ] Centralized Logs
- [ ] Basic Metrics

## Success Criteria

- Users can log in.
- Workers register successfully.
- Controller discovers all workers.
- Scheduler assigns workloads.
- Docker containers are deployed.
- Dashboard displays cluster status.
- Logs are collected.
- Metrics are visible.
